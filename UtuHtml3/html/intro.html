+ Conjuntos I


  • Los conjuntos se nombran en mayúsculas => A, B, C
  • Los ementos se nombran en minúsculas => a, b, c

  • FORMAS DE DEFINIR CONJUNTOS

    • Extensiva
      • Nombrar todos los elementos del conjunto
    • Comprensión
      • Nombrar la/s propiedades

  • EJEMPLOS
    1. A = {Ana, Juan, Daniel...}
    2. A = {x/x ∈ 1ºBL} ==> X tal que X pertenece a 1º BL
    3. X = {1, 2, 3, 4}
    4. X = (x/x ℕ y 1 < x < 4)
